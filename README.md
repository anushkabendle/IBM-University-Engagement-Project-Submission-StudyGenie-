# IBM-University-Engagement-Project-Submission-StudyGenie
> **Smart Genie** is an AI-powered Smart Study Assistant built using **IBM watsonx Orchestrate** that helps students plan, learn, and prepare for exams efficiently. It creates personalized study plans based on subjects, goals, exam dates, and available study time. Using **RAG** and a **multi-agent architecture**, Smart Genie retrieves trusted learning content, generates study materials, tracks progress, and provides intelligent learning assistance through a conversational AI interface.

---

## ✨ Features

- 🎯 Personalized study plans based on subjects, goals, exam dates, and available study hours
- 📅 Daily and weekly AI-generated study schedules
- 📖 RAG-powered content retrieval from trusted study materials
- 📝 AI-generated notes, summaries, flashcards, and revision guides
- ❓ Quiz and practice question generation with instant feedback
- 📊 Progress tracking with performance insights and completed topics
- 🤖 Interactive AI Learning Assistant for doubt solving and concept explanations
- ⏰ Smart reminders and adaptive revision planning
- 📱 Responsive and user-friendly interface

---

## 🏗️ Multi-Agent Architecture

Smart Genie consists of multiple AI agents working together:

### 📅 Study Planner Agent
- Creates personalized study schedules
- Optimizes study time based on goals and deadlines

### 📚 Content Retrieval Agent
- Uses RAG to fetch relevant information from trusted learning resources
- Summarizes educational content for quick understanding

### 📝 Quiz & Assessment Agent
- Generates quizzes and practice questions
- Evaluates performance and identifies weak areas

### 💬 Learning Assistant Agent
- Answers academic questions
- Explains difficult concepts
- Recommends personalized revision strategies

## ⚙️ How It Works

1. Students enter their subjects, learning goals, available study hours, and exam dates.
2. The Study Planner Agent creates a personalized study schedule.
3. The Content Retrieval Agent fetches relevant information from trusted study materials using RAG.
4. Smart Genie generates notes, summaries, flashcards, and quizzes.
5. The Quiz & Assessment Agent evaluates student performance.
6. The Learning Assistant provides explanations, revision strategies, and personalized recommendations.
7. The dashboard tracks progress and updates future study plans based on learning performance.
